# Python - Object Relational Mapping

## Description

This project introduces the concept of Object Relational Mapping (ORM) using Python.  
The goal is to connect Python scripts to a MySQL database and perform database operations.

The project is divided into two parts:

1. Using the **MySQLdb** module to execute SQL queries directly from Python.
2. Using **SQLAlchemy**, an ORM that allows developers to interact with the database using Python objects instead of writing SQL queries.

## Learning Objectives

At the end of this project, you should be able to explain:

- How to connect to a MySQL database from a Python script
- How to execute SELECT queries using Python
- How to INSERT data into a MySQL table using Python
- What Object Relational Mapping (ORM) means
- How to map a Python class to a MySQL table

## Technologies

- Python 3.8
- MySQL 8.0
- MySQLdb (mysqlclient)
- SQLAlchemy

## Concepts Covered

- Connecting Python to MySQL
- Executing SQL queries in Python
